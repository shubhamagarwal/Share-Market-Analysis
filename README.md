## Prerequisites

```
node (version > 10)
npm

```

## Tech Stack

```
React JS
NodeJS
Highcharts
Enzyme
chai
etc
```

# Run development project locally

```
Clone https://github.com/shubhamagarwal/Share-Market-Analysis.git
npm install
npm run build
npm run start
open http://localhost:3010 to view project on the browser
```


# Run Test Cases

```
npm run test
```
